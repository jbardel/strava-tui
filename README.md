# Strava TUI

Application terminal Rust utilisant Ratatui pour afficher les activités Strava.

## Fonctionnalités

- Authentification Strava
- Affichage des activités
- Navigation clavier
- Vue détaillée
- Interface terminal interactive

## Lancement

```bash
cp .env.example .env
cargo run
```

## Contrôles

- ↑ / ↓ : naviguer
- q : quitter
